Correlation corrected least squares

Accompanying files for:

Karl Fogelmark, Michael A. Lomholt, Anders Irbäck and Tobias Ambjörnsson Fitting a function to time-dependent ensemble averages
Submitted to Nature communications (2016)

  • ccls.tar.gz contains ready-to-use code for CCLS algorithm in Python, octave/matlab and common lisp, as well as example data. README file for instructions on how to use it.
  • gen_traj.tar.gz Contains the python scripts for generating the trajectories used in the article.
